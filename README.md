# 📦 Supply Chain Control Tower

An end-to-end Business Intelligence project that simulates a real-world Supply Chain Control Tower using **PostgreSQL**, **Power BI**, **DAX**, and **Python**.

The project focuses on inventory management, supplier performance, shipment monitoring, and operational insights through an interactive dashboard built on a star-schema data warehouse.

---

## 🚀 Project Objectives

* Design a Supply Chain Data Warehouse using a Star Schema.
* Analyze inventory, suppliers, purchase orders, shipments, and customer orders.
* Build interactive Power BI dashboards for business decision-making.
* Create business KPIs using DAX measures.
* Provide operational insights for supply chain performance.
